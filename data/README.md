# Data

Raw tweet files are not committed to this repository (Twitter/X ToS).

## How to get the data

1. ClimateMiSt: https://github.com/enric1994/climatemist
   → place files in data/raw/climatemist/

2. CMU-MisCov19: https://github.com/lamyiowce/MisCov19
   → place files in data/raw/cmu-miscov19/

3. MuMiN: https://mumin-dataset.github.io/
   → place files in data/raw/mumin/
